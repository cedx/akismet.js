_queryService

(
  • endPoint
  • fields
  • callback
)
private async

Queries the service by posting the specified fields to a given end point, and returns the response as a string.

Parameters:

  • endPoint String

    URL of the end point to query.

  • fields Object

    Fields describing the query body.

  • callback Function

    The callback to invoke at the end. It is passed two arguments:

    • error containing the error that occurred, if any.
    • response containing the response body returned by the remote service.

Properties

apiKey

String

blog

String

The front page or home URL of the instance making requests.

userAgent

String

The user agent string to use when making requests. If possible, the user agent string should always have the following format: Application Name/Version | Plugin Name/Version.

useSecureRequests

Boolean

A value indicating whether to use secure requests when querying the service database.

Default: false
